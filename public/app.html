ESP32 Devices
One sensor per keg in the cellar + one or more displays at the bar
🌡 CELLAR — Sensor ESP32s
One per keg. Reads flow meter, temperature, CO₂ pressure and FOB switch. Publishes to MQTT every second.
→
☁ SERVER
Receives all sensor data, calculates volumes, checks thresholds, updates database, sends alerts.
→
📺 BAR — Display ESP32s
One or two per restaurant. Shows all kegs cycling on TFT screen. Receives data from server via MQTT.
